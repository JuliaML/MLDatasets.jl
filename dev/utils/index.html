MLDatasets.read_planetoid_data — Function

Read any of the citation network datasets “Cora”, “CiteSeer” and “PubMed” from the “Revisiting Semi-Supervised Learning with Graph Embeddings” paper. Nodes represent documents and edges represent citation links.

Data collected from https://github.com/kimiyoung/planetoid/raw/master/data
